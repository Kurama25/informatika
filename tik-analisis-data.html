TIK & Analisis Data

📊 Teknologi Informasi & Komunikasi (TIK)

Pada jenjang Kelas X SMK, pemanfaatan TIK tidak lagi sekadar belajar mengetik, melainkan berfokus pada Integrasi Konten Aplikasi Perkantoran. Kamu diajarkan untuk menghubungkan berbagai data antar-aplikasi secara otomatis demi efisiensi kerja profesional.

Dua teknik integrasi data utama yang sering digunakan adalah:

  • Object Embedding: Menanamkan objek dari suatu aplikasi ke aplikasi lain. Jika file sumber diubah, objek di file tujuan tidak ikut berubah secara otomatis.
  • Object Linking: Menghubungkan data antar-aplikasi. Perubahan data pada file sumber (misal: tabel Excel) akan langsung memperbarui grafik di file tujuan (misal: Word atau PowerPoint) secara otomatis (real-time).

📈 Fondasi Analisis Data

Analisis Data adalah proses pengumpulan, transformasi, pembersihan, dan pemodelan data dengan tujuan menemukan informasi yang berguna, mendukung pengambilan keputusan, dan membuat kesimpulan yang tepat.

Proses dasar dalam mengolah data meliputi:

  • Pengumpulan Data: Mencari data mentah melalui survei, log sistem, atau observasi.
  • Pembersihan Data (Data Cleaning): Membuang data yang duplikat, kosong, atau tidak valid.
  • Analisis Korelasi & Statistik: Menggunakan rumus matematika (seperti Total/SUM, Rata-rata/AVERAGE, Nilai Tertinggi/MAX) untuk mencari makna dari kumpulan angka.
  • Visualisasi Data: Menyajikan data dalam bentuk grafik batang, garis, atau lingkaran agar lebih mudah dipahami oleh audiens dalam sekejap mata.
